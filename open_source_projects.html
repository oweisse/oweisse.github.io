SGX CryptoWrapper

A Python wrapper for sgx_tcrypto. The "sample_libcrypto" is marked not for production use, but the crypto code used inside the secure enclaves and the trusted libraries is golden (and licenced for re-use). The wrapped C++ code uses the production IPP crypto libraraies. Be aware that when using python scraping secrets from memory is not trivial.

Source: Coming soom..

PySignal

A Python wrapper for Signal-Hound API for Specturm-Analyzers (BB60C and SA124B) and Tracking-Generator (TG124A), including some basic FFT analysis code.

Source: https://github.com/oweisse/pysignal

mSaleaeLogic

A Matlab wrapper for Saleae logic analyzer network API.

Source: https://github.com/oweisse/msaleaelogic

DPA v4 Contest Submission

The DPA contest candidate as submitted. Includes the constraint solver described in CHES 2014, and decoders that were described in the HASP 2013 work

Source: https://github.com/oweisse/dpav4-contest
